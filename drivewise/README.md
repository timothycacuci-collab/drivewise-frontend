# DriveWise 🚗

AI-powered driving theory test practice app — 5 countries, 12 languages.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file:
   ```
   REACT_APP_BACKEND_URL=https://your-backend-url.com
   ```

3. Run locally:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Add environment variable: `REACT_APP_BACKEND_URL`
4. Click Deploy!

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router v6
- TanStack Query
- Axios
- Phosphor Icons
- Sonner (toasts)
